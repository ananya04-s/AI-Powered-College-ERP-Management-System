# 🎓 Campus360 AI – Intelligent College ERP Management System

![React](https://img.shields.io/badge/React-18-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)
![Supabase](https://img.shields.io/badge/Supabase-Backend-green)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-Database-blue)
![License](https://img.shields.io/badge/License-MIT-red)

Campus360 AI is a next-generation **AI-powered College ERP Management System** designed to digitize and automate academic and administrative operations within educational institutions. The platform provides role-based access for Students, Faculty, Parents, College Administrators, and Super Administrators through a modern, responsive, and secure web application.

By integrating AI-driven analytics, intelligent insights, and automation, Campus360 AI enhances academic management, improves decision-making, and simplifies everyday campus operations.
